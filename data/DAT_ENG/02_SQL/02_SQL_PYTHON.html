Connect to MySQL Database#

Mit einem sogenannten Conncetor kann Python mit der Datenbank verbunden werden. Dazu müssen die Koordinaten (IP, Port, Benutzer und Passwort) des MySQL-Servers bekannt sein. Weitere Informationen unter: https://realpython.com/python-mysql/

import mysql.connector

host = "192.168.1.64"
port = "3306"
user = "pi"
passwd = "pi"

mydb = mysql.connector.connect(
    host=host,
    port=port,
    user=user,
    passwd=passwd)

Create Table#

  1. Unsere neue Datenbank ist noch leer. Um diese mit Informationen zu füllen, erstellen wir eine neue Tabelle customer

  2. Die Tabelle wird mit 3 Attributen erstellt, id, name, address. id soll als primay-key eine fortlaufende, einzigartige nummer für jeden Eintrag erstellen.

  3. Die Attribute name und address sollen in Charakter-Arrays mit maximaler Länge von 255 gespeichert werden.

Select record from table#

  1. Um Daten aus der Datenbank-Tabelle auszulesen, können wir das SELECT-Statement mit * verbinden, wobei das * für alle steht.

sql = "SELECT * from customers"

mycursor.execute(sql)
myresult = mycursor.fetchall()

for x in myresult:
    print(x)

---------------------------------------------------------------------------
NameError                                 Traceback (most recent call last)

----> 3 mycursor.execute(sql)

  1. Um das Auslesen anzupassen, zum Beispiel nur ein Attribut aus der Tabelle customer, kann das * durch den/die Attribut-Namen ersetzt werden. Zum Beispiel SELECT name FROM customers

  2. Anstelle fetchall() kann auch fetchone() verwendet werden, um das erste Element der Ausgabe abzufragen.

  3. Eir können das Resultat mit dem Schlüsselwort WHERE Filtern. Beachten Sie, wie die Abfrage-Variabel in einer separaten Variabel gespeichert ist. Diese Technik nennt sich parametrisierte Abfrage und soll SQL-Injection verhindern.(SQL Injection Explained, SQL Injection Prevention Cheat Sheet)

sql = "SELECT * FROM customers WHERE address = %s"
adr = ("California", ) # Could not process parameters: str(California), it must be of type list, tuple or dict

mycursor.execute(sql, adr)

myresults = mycursor.fetchall()

for x in myresults:
    print(x)

  1. Jetzt wollen wir das join-Statement auf zwei Tabellen anwenden. Es gibt verschiedene Join-Statements. Das einfache JOIN, auch INNER JOIN, zeigt die Resultate an, welche im relevanten Attribut in beiden Tabellen vorkommt.

  2. Wir wollen eine Liste aller Kunden mit deren order_ids. Die verbindung zwischen Kunde und Bestellungen sind in der orders-Tabelle hinterlegt. Um dies zu erreichen nehmen wir die Kunden-Namen und Ids und verbinden diese mit den customer_id aus der Tabelle orders.

sql = "SELECT c.name, o.id FROM customers c JOIN orders o ON c.id = o.customer_id"
mycursor.execute(sql)

myresult = mycursor.fetchall()

for x in myresult:
