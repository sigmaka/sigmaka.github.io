Interaktive Plots#


← Einführung/Inhalt

Je nach Situation können Interaktive Plots helfen, eine Geschichte zu erzählen. Sie ermöglichen es, Daten selber zu erleben und zu erfahren. ACHTUNG: Dies kann zu angeregten Diskussionen führen, welche nicht immer Zielführend sind. Manchmal ist ein statischer Plot (z.B. mit Matplotlib) hilfreicher, da Sie selber bestimmen können, was der Empfänger sieht (und wie er es Interpretiert).

Ipywidgets#

Ipywidgets ermöglichen Interaktion innerhalb des Jupyter-Notebooks. Hier finden Sie weitere Informationen: Bring your Jupyter Notebook to life with interactive widgets.

  1. pip install ipywidgets

  2. Einschalten Ipywidgets in Jupyter Notebooks in einer Komandozeile mit
    jupyter nbextension enable –py widgetsnbextension

import matplotlib.pyplot as plt
import seaborn as sns
from ipywidgets import interact, interactive
import numpy as np

titanic = sns.load_dataset('titanic')

@interact
def create_fare_plot(col = titanic.drop(['fare', 'age'], axis=1).columns):
    sns.barplot(data=titanic, x=col, y='fare')
    plt.title(f'Bar-Plot Durchschnitt Fahrpreis gruppiert nach {col}')

def f(m, b):
    plt.figure()
    x = np.linspace(-10, 10, num=1000)
    plt.plot(x, m * x + b)
    plt.ylim(-5, 5)
    plt.show()

interactive_plot = interactive(f, m=(-2.0, 2.0), b=(-3, 3, 0.5))

Voila#

Voila ist ein Packet, welches ein Jupyter Notebook in ein Präsentierbares Web-App Dashboard verwandelt. Weitere Informationen finden Sie hier: Using Voilà to Turn Your Jupyter Notebook into a Web Application

  1. pip install voila

  2. Start Voila über den neuen Voila-Button in der Kommandozeile

Dash by Plotly#

Dash ist eine Toolbox zur Erstellung von Interaktiven Dashboards basierend auf Plotly. Weitere Informationen finden Sie hier: https://dash.plotly.com/tutorial

  1. pip install dash

import dash
from dash import dcc, html
import plotly.express as px
import pandas as pd
import seaborn as sns
app = dash.Dash()
df = sns.load_dataset('titanic')
fig = px.scatter(
                 df,
                 x="fare",
                 y="age",
                 size="pclass",
                 color="alive",
                 hover_name="embark_town",
                 log_x=True,
                 size_max=60
                )
app.layout = html.Div(children = [
                                  html.H1(children='Titanic Dashboard'),
                                  dcc.Graph(id="fare_vs_age", figure=fig)])

if __name__ == "__main__":
    app.run_server(debug=False)
