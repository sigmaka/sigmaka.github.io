Unterrichtsbeispiel#


← Einführung NumPy

Lernziele#

  • Sie kennen NumPy und können es korrekt importieren.

  • Sie erkennen, dass NumPy-Arrays gegenüber Listen deutliche Vorteile haben in Bezug auf mathematische/numerische Berechnungen.

  • Sie kennen die Gemeinsamkeiten & Unterschiede zwischen NumPy-Arrays und Listen.

  • Sie können einen 1D-Array erstellen und damit Berechnungen durchführen.

Aufgabe#

Sie stehen auf einem Turm mit einer Höhe von \(y_0 = 60 m\) und werfen einen Stein mit \(v_0 = 12.5 m/s\) in einem Winkel von \(\alpha = 40°\) in die Luft. Es soll die Flugbahn \(x(t)\) bzw. \(y(t)\) in Abhängigkeit der Zeit berechnet werden. Zum Vergleich soll die Aufgabe auf zwei verschiedene Arten gelöst werden.

  • a) mit den elementaren Python-Elementen (Datentyp list, Schleifen, range-Funktion etc.), d. h. ohne NumPy gelöst werden

  • b) mit dem “neuen” Datentyp NumPy-Array (ndarray)

Die Funktionen, die die Wurfbahn beschreiben (schiefer Wurf) beschreiben, kennen Sie aus der Physik (Kinematik), wobei \(g = -9.81 \, m/s^2\) die Erdbeschleunigung ist (negativ, weil sie nach unten wirkt).

\[x(t) = v_0 \cdot cos(\alpha) \cdot t\]
\[y(t) = \frac{1}{2} \cdot g \cdot t^2 + v_0 \cdot sin(\alpha) \cdot t + y_0\]
T = 5.0  # Zeitdauer [s]
dt = 0.02  # Zeitschrittweite [s]

g = -9.81  # Erdbeschl. [m/s^2]
y_0 = 60  # Starthöhe [m]
v_0 = 12.5  # Startgeschwindigkeit [m/s]
alpha = 40  # Abwurfwinkel [°]

a) Berechnung der Höhe des Steins für jeden Zeitschritt mit Python-Datentypen#

  • Erstellen Sie eine Liste t mit allen Zeitschritten von 0 bis T mit einer Schrittweite von dt

  • Erstellen Sie eine Liste x, worin für jeden Zeitschritt die horizontale Position (x) des Steins enthalten ist

  • Erstellen Sie eine Liste y, worin für jeden Zeitschritt die vertikale Position (y) des Steins enthalten ist

# a) mit Python-Listen:

import math as m  # wird benötigt für cos(), sin(), radians()

N = int(T/dt)  # Anzahl Zeitschritte

# t = []
# for i in range(N):
#   t.append(i*dt)

t = [i*dt for i in range(N)]  # kompaktere Alternative zu obigem Block (List Comprehension)
x = [v_0 * m.cos(m.radians(alpha)) * ti for ti in t]
y = [0.5 * g * ti**2 + v_0 * m.sin(m.radians(alpha)) * ti + y_0 for ti in t]

b) Berechnung wie in a), aber jetzt mit NumPy#

  • Importieren Sie NumPy

  • Erstellen Sie mit einer entsprechenden NumPy-Funktion den Zeitarray t gemäss Zeitdauer und Zeitschrittweite

  • Erstellen Sie einen Array x, worin wiederum für jeden Zeitschritt die horizontale Position (x) des Steins enthalten ist

  • Erstellen Sie einen Array y, worin wiederum für jeden Zeitschritt die vertikale Position (y) des Steins enthalten ist

# b) mit NumPy-Arrays:

import numpy as np  # enthält auch alle mathematischen Funktionen etc.

t = np.arange(0, T, dt)
x = v_0 * np.cos(np.radians(alpha)) * t
y = 0.5 * g * t**2 + v_0 * np.sin(np.radians(alpha)) * t + y_0

c) Fazit#

Vergleichen Sie Umsetzung in a) und b). Was fällt auf bzw. was sind Vorteile von NumPy? Notieren Sie nachfolgend ein kurzes Fazit.

Fazit: NumPy-Arrays sind für mathematische Berechnungen deutlich besser geeignet als Listen, da die Multiplikation, Addition etc. automatisch für alle Elemente im Array durchgeführt wird (und man daher keine Schleifen usw. benötigt). Der Code wird damit viel leserlicher und effizienter. NumPy enthält auch alle mathematischen Funktionen und damit wird das math-Modul nicht mehr benötigt. (Wie wir später noch sehen werden, haben Arrays auch in Bezug auf die Indizierung grosse Vorteile gegenüber Listen.)

d) Zeitpunkt der Landung#

  • Bestimmen Sie anhand der beiden Arrays t und y den ungefähren Zeitpunkt, bei dem der Stein am Boden aufschlägt und geben Sie diesen Zeitpunkt in der Form Der Stein landet nach ca. ... s am Boden. auf dem Bildschirm aus, wobei die Zeit auf 1 Nachkommastelle gerundet werden soll

  • Setzen Sie negative Werte im y-Array auf 0

  • Der x-Array soll ab dem Zeitpunkt des Aufschlages keine Änderung mehr aufweisen (konstanter Wert des Aufschlages für alle weiteren Zeitschritte)

# Bedingung für das Auftreffen am Boden: y(t) ist 0 (der Wert 0 wird aber kaum exakt erreicht, 
# d.h. wir können nicht direkt nach einer 0 im y-Array suchen)
# Idee: Wir bestimmen den Wert bzw. den Index des Wertes, wo y(t) möglichst nahe bei 0 ist. Dies
# lässt sich erreichen, wenn wir den Betrag von y bestimmen und dann das Minimum bzw. der Index
# des Minimums bestimmen.

ind = np.absolute(y).argmin()  # Betrag und dann Index des Minimums bestimmen (Die Funktion min()
# würde das Minimum selbst bestimmen, wir wollen aber den Index des Minimums.)
print("Der Stein landet nach ca. {} s am Boden.".format(round(t[ind],1)))  # Aufschlags-Zeitpunkt

y[y<0] = 0  # negative y-Werte auf 0 setzen (für Plot)
x[ind:] = x[ind]  # x-Werte nach Aufschlag auf Auftreffposition setzen (für Plot)

# .. = np.delete(x, ...)  # alternativ könnten x- und y-Array auch "abgeschnitten" werden

Der Stein landet nach ca. 4.4 s am Boden.

e) Grafische Darstellung der Flugbahn#

Die Library für grafische Darstellungen (matplotlib) werden wir in einem zukünftigen Unterrichtsblock genauer anschauen. Damit wir trotzdem eine Visualisierung der oben berechneten Flugbahn darstellen können, verwenden wir die Plot-Library an dieser Stelle ohne näher darauf einzugehen.

  • Aktivieren Sie die Plot-Zeile im untenstehenden Codeabschnitt.

  • Gehen Sie jede Zeile für sich durch. Können Sie nachvollziehen, was gemacht wird?

import matplotlib.pyplot as plt  # Importieren der Plotting-Library

plt.figure(figsize=(12,5))
plt.plot(x, y, 'r')
plt.xlabel("Distanz x [m]")
plt.ylabel("Höhe y [m]")
plt.title("Flugbahn")
plt.grid()
plt.show()
