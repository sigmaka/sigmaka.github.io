Unterrichtsbeispiel#


← Einführung Matplotlib

Lernziele#

  • Sie können die Plotting-Library Matplotlib korrekt importieren und anwenden.

  • Sie können ein Diagramm sowohl mit dem Pyplot-Interface als auch mit dem Axes-Interface erstellen.

  • Sie wissen, dass ein Diagramm immer vollständig und korrekt beschriftet sein muss und kennen die nötigen Matplotlib-Funktionen dazu.

  • Neben Liniediagrammen können Sie auch Balkendiagramme, Kuchendiagramme, Boxplots und Histogramme mit Matplotlib erstellen.

Aufgabe#

Wir werden ein einfaches Diagramm zuerst mit dem impliziten Pyplot-Interface und danach mit dem expliziten Axes-Interface erstellen, damit der Unterschied zwischen diesen beiden Interfaces von Matplotlib klar wird. Das erste Diagramm wird ein einfaches Liniendiagramm sein (also etwas, was sicherlich alle bereits einmal z. B. in Excel gemacht haben).

Wir wollen anhand eines einfachen Datensatzes Kennlinie.txt, den wir mit NumPy importieren, ein vollständiges und ansprechendes Diagramm mit Matplotlib erstellen, wobei der Fokus dabei ausnahmsweise nicht auf den Daten, sondern auf der Diagrammerstellung liegen soll. Im Textfile ist die experimentell ermittelte Kennlinie eines regelungstechnischen Systems (Regelstrecke) enthalten, wobei x_e dem Eingangssignal und x_a_raw dem Ausgangssignal bzw. dem linearisieren Ausgangssignal x_a_lin entspricht. Die Kennlinie soll grafisch dargestellt und die Abbildung exportiert werden (um sie beispielsweise in einem Bericht verwenden zu können).

a) Importieren der Textdatei#

  • Importieren Sie Matplotlib und NumPy.

  • Importieren Sie die Daten der Textdatei Kennlinie.txt im Ordner data und weisen Sie diese einem Array mit dem Variablennamenkennlinie zu.

import numpy as np
import matplotlib.pyplot as plt

kennlinie = np.loadtxt('data/Kennlinie.txt', skiprows=2, delimiter='\t')

pwm = kennlinie[:,0]  # Eingangssignal x_e (hier pwm genannt)
n_mess = kennlinie[:,1]  # Drehzahlmessung x_a_raw (hier n_mess genannt)
n_lin = kennlinie[:,2]  # Linearisiertes Drehzahlsignal x_a_lin (hier n_lin genannt)

b) Erstellen des Kennliniendiagramms mit dem Pyplot-Interface#

  • Erstellen Sie das Kennliniendiagramm mit dem Pyplot-Interface. Die Messpunkte sollen als rote Punkte (Kreise) dargestellt werden und mit gestrichelter Linie verbunden verbunden werden.

  • Stellen Sie zusätzlich den linearisierten Verlauf als blaue Linie dar.

  • Beschriften Sie die Achsen korrekt, setzen Sie einen Titel (Kennliniendiagramm) und fügen Sie eine Legende (Rohdaten, Linearisierung) hinzu.

  • Aktivieren Sie die Gitternetzlinien.

  • Exportieren Sie das Diagramm als SVG-Datei Kenninie.svg in der Ordner img.

plt.figure(figsize=(10, 5))
plt.plot(pwm, n_mess, 'or--', label='Rohdaten')
plt.plot(pwm, n_lin, 'b', label='Linearisierung')
plt.title('Kennliniendiagramm')
plt.xlabel('PWM [%]')
plt.ylabel('Drehzahl [rpm]')
plt.grid()
plt.legend()
plt.savefig('img/Kennlinie.svg')
plt.show()

c) Erstellen des Kennliniendiagramms mit dem Axes-Interface#

  • Erstellen Sie dasselbe Diagramm wie oben, doch verwenden Sie nun übungshalber das Axes-Interface.

fig, ax = plt.subplots(figsize=(10, 5))
ax.plot(pwm, n_mess, 'or--', label='Rohdaten')
ax.plot(pwm, n_lin, 'b', label='Linearisierung')
ax.set_title('Kennliniendiagramm')
ax.set_xlabel('PWM [%]')
ax.set_ylabel('Drehzahl [rpm]')
ax.grid()
ax.legend()
plt.show()

d) Andere Diagrammtypen#

In einer kleinen Gruppenübung wollen wir andere Diagrammtypen anschauen, die wir in der Technik etwas weniger häufig verwenden. Holen Sie sich dabei Hilfe bei den Online-Beispielen von Matplotlib bzw. lassen Sie sich dort inspirieren. Verwenden Sie nach Möglichkeit das Axes-Interface und beschriften das Diagramm vollständig.

  • Balkendiagramm (Gruppe 1) der Hauptsprachen in der Schweiz

  • Kuchendiagramm (Gruppe 2) der Parteistärken in der Schweiz

  • Boxplot (Gruppe 3) einer Stichprobe

  • Histogramm (Gruppe 4) einer Stichprobe

Gruppe 1: Hauptsprachen in der Schweiz#

Erzeugen Sie ein Balkendiagramm für die in der Schweiz gesprochenen Sprachen.

d1_sprachen = ['Deutsch', 'Französisch', 'Italienisch', 'Rätoromanisch', 'Andere']
d1_anteile = [62.0, 22.7, 8.2, 0.5, 22.7]  # [%]

fig, ax = plt.subplots(figsize=(8,4))
ax.bar(d1_sprachen, d1_anteile)
ax.set_title('Hauptsprachen in der Schweiz')
ax.set_xlabel('Sprache')
ax.set_ylabel('Anteil [%]')
ax.grid()
plt.show()

Gruppe 2: Parteistärken in der Schweiz#

Erzeugen Sie ein Kuchendiagramm für die Parteistärken in der Schweiz.

d2_parteien = ['FDP', 'Mitte', 'SP', 'SVP', 'GLP', 'GRÜNE', 'Übrige']
d2_anteile = [14.25, 14.06, 18.27, 27.93, 7.55, 9.78, 8.16]  # [%]
d2_colors = ['#3872B5', '#D6862B', '#F0554D', '#4B8A3E', '#C4C43D', '#84B547', '#B8B8B8']

fig, ax = plt.subplots(figsize=(6,6))
ax.pie(x=d2_anteile, labels=d2_parteien, autopct='%.1f %%', colors=d2_colors)
ax.set_title('Parteistärken in der Schweiz')
ax.grid()
plt.show()

Gruppe 3: Boxplot einer Stichprobe von 10’000 Messungen#

Ein Bauteil wurde auf zwei verschiedenen Maschinen gefertigt und Sie haben jeweils 5000 Bauteile gemessen (Stichprobe). Vergleichen Sie die Qualität der beiden Maschinen mittels Boxplot.

d3_durchmesser = np.loadtxt('data/Stichprobe.csv', delimiter=';', skiprows=1)
d3_labels = ['Maschine 1', 'Maschine 2']

fig, ax = plt.subplots(figsize=(8,4))
ax.boxplot(d3_durchmesser, labels=d3_labels)
ax.set_title('Stichprobenvergleich via Kenngrössen')
ax.set_ylabel('Durchmesser [mm]')
ax.grid()
plt.show()

Gruppe 4: Histogramm einer Stichprobe von 10’000 Messungen#

Ein Bauteil wurde auf zwei verschiedenen Maschinen gefertigt und Sie haben jeweils 5000 Bauteile gemessen (Stichprobe). Vergleichen Sie die Qualität der beiden Maschinen mittels Histogramm.

d4_durchmesser = np.loadtxt('data/Stichprobe.csv', delimiter=';', skiprows=1)
d4_labels = ['Maschine 1', 'Maschine 2']

fig, ax = plt.subplots(figsize=(8,4))
ax.hist(d4_durchmesser, bins=20)
ax.set_title('Stichprobenvergleich via Durchmesserverteilung')
ax.set_xlabel('Durchmesser [mm]')
ax.set_ylabel('Häufigkeit')
ax.legend(d4_labels)
ax.grid()
plt.show()
