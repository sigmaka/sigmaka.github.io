3. RGB-Matrix#


← Übersicht

Den Begriff Matrix haben Sie sicherlich schon in unterschiedlichen Zusammenhängen gehört. Im mathematischen/programmiertechnischen Kontext bedeutet Matrix eine zeilen- bzw. spaltenweise Anordnung von Elementen. Entscheidend ist, dass eine Matrix in diesem Zusammenhang ein zweidimensionales Objekt ist und via Zeilen- bzw. Spaltenindex auf ein individuelles Element zugegriffen wird. Eine Matrix mit \(n\) Zeilen und \(m\) Spalten wird auch als \(n \times m\)-Matrix bezeichnet.

Wie Sie bereits wissen, beginnen die Indices in Python bei 0. Wenn wir z. B. umgangssprachlich von der ersten Zeile (oder Spalte) sprechen, entspricht dies jeweils der Zeile (oder Spalte) mit dem Index 0 (nicht 1).

3.1 Verschachtelte Liste#

Im Grundumfang von Python existiert kein expliziter Matrix-Datentyp, doch mittels einer Liste kann man ein Matrix-ähnliches Gebilde erzeugen. Da Listen selbst wiederum Listen enthalten können, kann man eine Matrix bilden, indem z. B. das erste Element der Liste die erste Zeile der Matrix repräsentiert und daher selbst eine Liste ist, welche die Spalten repräsentiert. Eine solche Liste wird verschachtelte Liste genannt, da sie als Elemente wiederum Listen enthält. Das folgende Beispiel soll den Sachverhalt veranschaulichen anhand einer “Matrix” mit 3 Zeilen und 2 Spalten mit den Elementen 1 bis 6.

\[\begin{split} \begin{bmatrix} 1 & 2\\ 3 & 4\\ 5 & 6\\ \end{bmatrix} \end{split}\]
# Bsp. 3x2-Matrix
matrix = [[1, 2], [3, 4], [5, 6]]  # 3x2-Matrix als verschachtelte Liste
matrix

[[1, 2], [3, 4], [5, 6]]

Um ein individuelles Element anzusprechen, wird bei einer verschachtelten Liste für jedes Verschachtelungslevel ein Index verwendet, der wie gehabt in eckigen Klammern steht. Mit dem ersten Index wird also die “oberste” Ebene der Liste angesprochen und mit dem zweiten Index die “zweitoberste” etc. Fassen wir die verschachtelte Liste als Matrix auf, so können wir den ersten Index als Zeilenindex und den zweiten Index als Spaltenindex auffassen.

matrix[0]  # erster Index = 0, d.h. "erste Zeile"

[1, 2]

matrix[1][1]  # erster Index = 1 ("zweite Zeile") und zweiter Index = 1 ("zweite Spalte")

4

Betrachten wir nun unsere RGB-Matrix mit 8 Zeilen und 8 Spalten, d. h. gesamthaft 64 LEDs, können wir diese 64 LEDs in einer verschachtelten Liste mit 8 Unterlisten (Zeilen) à 8 Elemente (Spalten) repräsentieren. Jedes Element dieser verschachtelten Liste entspricht der Farbe der LED (im RGB-Farbraum), d. h. besteht wiederum aus 3 Elementen r, b, g. Wir haben also gesamthaft drei Verschachtelungsebenen. Im ersten Beispiel (vgl. Einführung RGB-Matrix) haben wir das bereits gesehen.

rgb.rgb_matrix[zeile][spalte] = [rot, gruen, blau]  # Farbzuweisung

Das rgb_matrix-Attribut ist also eine verschachtelte Liste mit drei Ebenen, d. h. maximal sind drei Indices möglich.

rgb.rgb_matrix[3][4][2]  # Blau-Anteil der LED in der vierten Zeile und fünften Spalte

Da wir bei allen Aufgaben mit verschachtelten Listen arbeiten, ist es sehr wichtig, die Listenstruktur und Indizierung gut verstanden zu haben. Als Hilfestellung (u. a. zum Ausdrucken) kann die folgende Vorlage verwendet werden.
