8.3 Vererbung (Inheritance)#


← Übersicht

Ganz allgemein kann man sagen, dass die Vererbung eine Beziehung zwischen einer allgemeinen Klasse (Oberklasse oder Basisklasse) und einer spezialisierten Klasse (Unterklasse oder Subklasse) definiert. Die Vererbung ermöglicht es, die Attribute/Methoden einer existierenden Klasse in eine andere Klasse zu “kopieren” (vererben) und diese zu erweitern/ändern. Die Vererbung ist daher sehr hilfreich, um bestehenden Code weiterzuverwenden (Reusability).

Nachfolgend wird eine sehr einfache Klasse Person mit den Attributen vorname, name und geburtstag erstellt. Eine solche Klasse sollten Sie mit Ihrem jetzigen Wissen selbstständig implementieren können.

class Person:
    def __init__(self, vorname, nachname, geburtstag):
        self.vorname = vorname
        self.nachname = nachname
        self.geburtstag = geburtstag

    def __str__(self):
        return self.vorname + ' ' + self.nachname + ', ' + self.geburtstag

Nun wollen wir eine zweite Klasse Angestellter erstellen. Da ein Angestellter immer auch eine Person ist, d. h. einen Vor- und Nachnamen sowie ein Geburtsdatum hat, wäre es schön, wir könnten die Funktionalität der (allgemeinen) Klasse Person in die (spezifische) Klasse Angestellter vererben. Dies wird durch die Vererbung ermöglicht und ist im nachfolgenden Beispiel gezeigt. Alle Attribute der Oberklasse Person stehen damit in der Klasse Angestellter zur Verfügung, d. h. es muss nur noch das zusätzliche Attribut personalnummer erstellt werden.

class Angestellter(Person):  # Klasse Angestellter erbt von Klasse Person
    def __init__(self, vorname='', nachname='', geburtstag='', personalnummer=''):
        super().__init__(vorname, nachname, geburtstag)  # __init__-Methode der Oberklasse aufrufen
        # Person.__init__(self, vorname, nachname, geburtstag)  # alternativ
        self.personalnummer = personalnummer

    def __str__(self):
        return super().__str__() + ' (' + self.personalnummer + ')'
        # return Person.__str__(self) + ' (' + self.personalnummer + ')'

angestellter = Angestellter(vorname='Homer', nachname='Simpson', geburtstag='09.08.1969', personalnummer='007')
print(angestellter)

Homer Simpson, 09.08.1969 (007)

Zur Illustrierung dient wiederum das Klassendiagramm, welches die beiden Klassen zeigt und die Vererbung durch einen Pfeil (in Richtung der Basisklasse) visualisiert.

8.3.1 Mehrfachvererbung (Multiple Inheritance)#

Eine abgeleitete Klasse kann auch von mehreren Basisklassen erben. Dafür werden einfach mehrere Basisklassen in den Klammern hinter dem Klassennamen angegeben (durch Komma getrennt). Diese Reihenfolge entspricht auch der Suchreihenfolge, d.h. eine Methode wird v.l.n.r. durch die Basisklassen gesucht.

class Unterklasse(Basisklasse1, Basisklasse2, Basisklasse3, ...):
    ...

Mehrfachvererbungen werden allerdings rasch unübersichtlich und sollten daher möglichst vermieden werden.
