Lernziele#

  • Kennenlernen und Anwenden von SciPy anhand eines typischen Beispiels in der Signalverarbeitung.

  • Filtern eines verrauschten Signals.

  • Numerische Ableitung/Intergration eines gemessenen Signals.

Aufgabe#

Für Filmaufnahmen wird ein Drohnenflug durchgeführt, bei dem die Drohne vertikal aufsteigt und sich gleichzeitig um ihre Achse dreht. Dabei wird die Höhe \(h\) [m] und die Winkelgeschwindigkeit \(\omega\) [rad/s] aufgezeichnet. In nachfolgender Abbildung ist das betrachtete System abgebildet.

Die Aufgabe besteht darin, die folgenden Grössen aus den Messdaten zu bestimmen:

  • Geschwindigkeitsverlauf (vertikal) \(v(t)\) aus Höhe \(h(t)\)

  • Winkelverlauf \(\phi(t)\) aus Winkelgeschwindigkeit \(\omega(t)\)

import numpy as np
import matplotlib.pyplot as plt
from scipy import signal, integrate  # SciPy-Module importieren

a) Datensatz einlesen und plotten#

Lesen Sie das Textfile Drohne.csv ein und weisen Sie die Daten spaltenweise den drei Arrays t (Zeit), h (Höhe) sowie omega (Winkelgeschw.) zu. Stellen Sie die beiden Messignale (Höhe bzw. Winkelgeschwindigkeit) anschliessend grafisch dar.

data = np.loadtxt('data/Drohne.csv', skiprows=3, delimiter=';')
t, h, omega = data.T

fig, axs = plt.subplots(figsize=(10,7), nrows=2)  # mehrere Subplots mit nrows = ... und/oder ncols = ...
axs[0].plot(t, h, 'b')
axs[0].set_ylabel('Höhe $h$ [m]')
axs[0].grid()
axs[1].plot(t, omega, 'k')
axs[1].set_xlabel('Zeit $t$ [s]')
axs[1].set_ylabel('Winkelgeschwindigkeit $\omega$ [rad/s]')
axs[1].grid()
plt.show()

b) Geschwindigkeitsverlauf aus Höhen-Messdaten#

Bei dieser Aufgabe geht es darum, den vertikalen Geschwindigkeitsverlauf \(v(t)\) aus den Höhen-Messdaten \(h(t)\) zu bestimmen. Wir möchten grob die maximale Geschwindigkeit beim Aufsteigen der Drohne bestimmen. Wissen Sie noch, wie Geschwindigkeit und Position ganz generell zusammenhängen?

# Signal muss gefiltert werden vor Ableitung, da Ableitung (Differenzierung) sehr sensitiv auf Rauschen
b, a = signal.butter(2, 0.75, btype='lowpass', fs=50)  # Butterworth-Tiefpassfilter
h_filt = signal.filtfilt(b, a, h)  # Filter anwenden

#v = np.gradient(h, t)  # Ableitung des ungefilterten Signals (unbrauchbar wegen Rauschen)
v = np.gradient(h_filt, t)  # Ableiten -> Geschwindigkeitsverlauf

plt.figure(figsize=(10,3))
plt.plot(t, v, 'r')
plt.xlabel('Zeit $t$ [s]')
plt.ylabel('Geschwindigkeit $v$ [m/s]')
plt.grid()
plt.title('Geschwindigkeitsverlauf')
plt.show()

c) Winkelverlauf aus Winkelgeschwindigkeit-Messdaten#

Nach der obigen Teilaufgabe dürften Sie nun auch wissen, wie der Winkelverlauf \(\phi(t)\) aus der Winkelgeschwindigkeit \(\omega(t)\) bestimmt werden kann. Stellen Sie den Winkelverlauf in [°] über der Zeit dar.

phi = integrate.cumulative_trapezoid(omega, t, initial=0)  # Intergrieren -> Winkelverlauf

plt.figure(figsize=(10, 3))
plt.plot(t, np.degrees(phi), 'g')
plt.xlabel('Zeit $t$ [s]')
plt.ylabel('Winkel $\phi$ [°]')
plt.grid()
plt.title('Winkelverlauf')
plt.show()
