Signalverarbeitung#


← Modulübersicht

SciPy (Scientific Python) erweitert den Umfang von NumPy um viele fundamentale Algorithmen für die Singalverarbeitung, die Optimierung, das Lösen von Gleichungen, etc. Die wichtigsten Module von SciPy sind:

  • Signalverarbeitung (Signal Processing) - scipy.signal

  • Integration - scipy.integrate

  • Optimierung (Optimization) - scipy.optimize

  • Interpolation - scipy.interpolation

  • Statistik (Statistics) - scipy.stats

  • Frequenzanalyse (Fourier Transforms) - scipy.fft

  • …

Das Paket werden wir hier nur grob überfliegen, da wir davon dann vor allem das Modul scipy.fft verwenden im Rahmen der Frequenzanalyse (vgl. FFT). Da SciPy viele andere, nützliche Funktionen enthält, lohnt sich eine kurze Übersicht aber auf jeden Fall. Mittlerweile sollte es Ihnen nicht mehr schwer fallen, eine bestimmte Funktion aus einem bestimmten Paket/Unterpaket anhand der Dokumentation/Hilfe selbstständig einsetzen zu können.

1. Motivation#

SciPy ist eine Sammlung von mathematischen Algorithmen und Funktionen (basierend auf NumPy) für verschiedene Spezialgebiete. Es erweitert damit unsere “Toolbox” um wichtige Werkzeuge.

  • SciPy User Guide

  • SciPy Reference

1.1 Import#

Von SciPy werden üblicherweise jeweils die entsprechenden Module importiert. Mit z. B. import scipy as sp kann selbstverständlich auch das gesamte Paket importiert werden.

import numpy as np
import matplotlib.pyplot as plt
from scipy import constants, signal, integrate, optimize, interpolate, stats, fft

2. Nützliche Funktionen (Auswahl)#

Nachfolgend sind nützliche Funktionen aufgelistet, die in vielen praktischen Problemstellungen sehr hilfreich sein können.

2.1 Konstanten (scipy.constants)#

  • Diverse math./phys. Konstanten

print(constants.c)  # Lichtgeschwindigkeit (alternativ: .speed_of_light) [m/s]
print(constants.mach)  # Schallgeschwindigkeit (alternativ: .speed_of_sound) [m/s]
print(constants.g)  # Erdbeschleunigung [m/s^2]

340.5
9.80665

2.2 Signalverarbeitung (scipy.signal)#

  • medfilt() - Median Filter

  • savgol_filter() - Polynomfilter

  • butter() - Butterworth-Filter (Tief-/Hoch-/Bandpass)

  • filtfilt() - Filter anwenden (vgl. auch .lfilter())

  • detrend() - Trend entfernen

  • decimate() - Downsampling mit Anti-Aliasing-Filter

  • resample() - Resampling

  • chirp() - Chirp-Signal

  • sawtooth() - Sägezahnsignal

  • square() - Rechtecksignal

  • get_window() - Fensterfunktion für Frequenzanalyse, vgl. FFT

  • find_peaks() - Peaks in einem Signal detektieren

  • …

y = 0.1 * np.random.randn(100)  # Messdaten simulieren
y[57] = 1  # Ausreisser
y_medfilt = signal.medfilt(y, 9)  # Medianfilter mit Fensterlänge 9
y_polfilt = signal.savgol_filter(y, window_length=9, polyorder=3)  # Polynomfilter (kubisch)
b, a = signal.butter(N=2, Wn=0.2, btype='low')  # Butterworth-Tiefpassfilter
y_filtfilt = signal.filtfilt(b, a, y)  # Butterworth-Filter anwenden

plt.figure(figsize=(12, 5))
plt.plot(y, 'b:', label='raw')
plt.plot(y_medfilt, 'm', label='filtered (median)')
plt.plot(y_polfilt, 'k', label='filtered (polynomial)')
plt.plot(y_filtfilt, 'r', label='filtered (butterworth)')
plt.legend()
plt.show()

2.3 Integration (scipy.integrate)#

Das Integrieren (und das Ableiten als Gegenstück) spielt in vielen Bereichen eine wichtige Rolle. Im Gegensatz zum analytischen Integrieren/Ableiten, das z. T. mit grossem Aufwand verbunden ist, ist das numerische Integrieren/Ableiten äusserst unkompliziert. Das Integral kann intuitiv als Fläche unter einer Kurve verstanden werden, was die folgende Abbildung illustriert.

Betrachtet man die obige Abbildung und überlegt sich, wie man die Fläche unter der Kurve anhand der “Messpunkte” (schwarze Punkte) annähern könnte, so landet man rasch bei der sogenannten Trapezregel, d. h. die Fläche kann in vertikale Streifen unterteilt werden und diese aufsummiert werden. Das Integral wird damit zu einer Summe und kann ohne viel Aufwand berechnet werden.

\[\int y(x) \,dx \approx \sum_{i} \frac{y_i + y_{i-1}}{2} \cdot \Delta x\]

In SciPy stehen dafür die folgenden Funktionen bereit:

  • trapezoid() - num. Integration mit Trapezregel

  • cumulative_trapezoid() - num. Integration mit Trapezregel (kumulativ)

  • …

y = [1, 3, 3, 2, 5, 2, 6, 2, 1, 5, 1]
x = np.linspace(0, 5, len(y))
integrate.trapezoid(y, x)  # numerisches Integral gemäss Trapezregel

15.0

integrate.cumulative_trapezoid(y, x)  # numerisches Integral gemäss Trapezregel (kumuliert)

array([ 1.  ,  2.5 ,  3.75,  5.5 ,  7.25,  9.25, 11.25, 12.  , 13.5 ,
       15.  ])

Ableitung#

Das Gegenstück zur Integration ist die Ableitung. Die Ableitung kann intuitiv als Steigung der Kurve verstanden werden.

Auch die Ableitung kann numerisch sehr einfach berechnet werden, nämlich mit dem altbekannten “Steigungsdreieck”. Die Ableitung (Differenzialquotient) wird damit zu einem einfachen Differenzenquotienten.

\[ \frac{dy(x)}{dx} \approx \frac{y_{i+1} - y_{i-1}}{2 \cdot \Delta x} \]

Zur Berechnung der Ableitung wird am besten die Funktion gradient() von NumPy verwendet. (Der Begriff Gradient ist ein anderes (allgemeineres) Wort für Steigung.)

np.gradient(y, x)  # Ableitung (mit NumPy)

array([ 4.,  2., -1.,  2.,  0.,  1.,  0., -5.,  3.,  0., -8.])

2.4 Optimierung bzw. Gleichungslöser (scipy.optimize)#

  • minimize_scalar() - Minimisierung einer Funktion mit einer Variable

  • minimize() - Minimisierung einer Funktion mit mehreren Variablen

  • root_scalar() - Lösen einer skalaren Funktion

  • root() - Lösen einer Vektorfunktion

  • least_squares() - Nichtlineare Optimierung mit Randbedingungen

  • curve_fit() - Ausgleichsrechnung, beliebige nichtlineare Funktion

Damit können u. a. Gleichungen bzw. Gleichungssysteme gelöst oder minimiert werden.

def f(x):
    return 0.01*x**2 + x - 1  # Gleichung (Funktion), die gelöst werden soll

sol = optimize.root_scalar(f, x0=0)  # eine der Lösungen der quadratischen Gleichung  f(x)
sol.root  # Nullstelle (x-Wert)

0.9901951359278484

sol = optimize.minimize_scalar(f)  # Minimum (Scheitelpunkt) der obigen Funktion f(x)
sol.x, sol.fun  # x- bzw. Funktionswert des Minimums

(-50.000000000000355, -26.0)

Mit curve_fit() kann ein beliebiges Modell (Funktion) an Daten “gefittet”, d. h. an Daten angepasst werden. Dies könnte z. B. verwendet werden, um eine gemessene Sprungantwort eines P-T1-Gliedes zu identifizieren (siehe nachfolgendes Beispiel).

t = np.linspace(0, 1, 100)  # Zeitarray
x_a = 3.5 * (1-np.exp(-t/0.15)) + 0.1*np.random.randn(100)  # simulierte Daten (wären in Realität Messdaten)

def f(t, K_P, T_1):
    return K_P * (1-np.exp(-t/T_1))  # Modell (Funktion) eines P-T1-Gliedes

popt, pcov = optimize.curve_fit(f, t, x_a)  # Modell wird "gefittet"
 
plt.figure(figsize=(12, 5))
plt.plot(t, x_a, '.b:', label='Messdaten')
plt.plot(t, f(t, *popt), 'r', label='Modell (P-T1)')
plt.text(0.0, 3.0, "$K_P$ = {:.2f}, $T_1$ = {:.2f} s".format(*popt))
plt.legend()
plt.show()

2.5 Interpolation (scipy.interpolate)#

  • CubicSpline() - Kubische Interpolation

  • BSpline() - B-Spline-Interpolation

  • …

Für die lineare Interpolation wird am besten die Funktion interp() von NumPy verwendet.

spline = interpolate.CubicSpline(x, y)
x_interp = np.linspace(0, max(x), 1000)
y_interp = spline(x_interp)  # Spline-Interpolation

y_lin_interp = np.interp(x_interp, x, y)  # lin. Interpolation

plt.figure(figsize=(12, 5))
plt.plot(x, y, 'ob', label='Messdaten')
plt.plot(x_interp, y_interp, 'r--', label='Spline-Interpolation')
plt.plot(x_interp, y_lin_interp, 'g--', label='Lineare Interpolation')
plt.legend()
plt.show()

2.6 Statistik (scipy.stats)#

In diesem Modul sind nützliche Funktionen für statistische Analysen (statistische Grössen, Verteilungen, Hypothesentests etc.). Wir werden dieses Modul nicht weiter einsetzen.

  • linregress() - lineare Regression (Gerade in Daten “fitten”)

  • pearsonr() - Korrelationskoeffizient

  • …

Die Funktion linregress() wird häufig verwendet, um ein lineares Modell (Gerade) an Messdaten zu fitten. Im folgenden Beispiel wird dies illustriert.

x = np.arange(10)
y = 1.5 + 0.75 * x + np.random.randn(10)  # Messdaten simulieren (Gerade + Rauschen)

res = stats.linregress(x, y)  # lineare Regression
y_fit = res.intercept + res.slope * x  # Modell

plt.figure(figsize=(12, 5))
plt.plot(x, y, 'ob--', label='Messdaten')
plt.plot(x, y_fit, 'r', label='Regression')
plt.text(0.0, 5.0, "Steigung = {:.2f}, Offset = {:.2f}".format(res.slope, res.intercept))
plt.legend()
plt.show()
