Workflow#


← Einführung

Der Ablauf der Berechnung des Frequenzspektrums ist immer gleich, d. h. er kann gemäss den folgenden Schritten durchgeführt werden:

  1. Zeitsignal y bereitstellen und Abtastfrequenz fs bzw. Sampleanzahl N bestimmen

  2. Fensterfunktion anwenden:

    • Fensterfunktion erzeugen w = signal.windows.get_window('...', N)

    • Fensterfunktion anwenden (Offset beachten) y_w = w * (y - y.mean()) + y.mean()

  3. FFT berechnen mittels Y = fft.rfft(y_w) und Amplituden korrekt skalieren

    • Amplituden bestimmen A = abs(Y)/N

    • Amplituden skalieren A[1:] = A[1:]*2

  4. Frequenzachse erzeugen mittels f = fft.rfftfreq(N, 1/fs)

  5. Amplitudenspektrum grafisch darstellen

Beispiel#

Anhand eines Beispieldatensatzes (data/messung.csv) wollen wir den FFT-Workflow durchgehen:

import numpy as np
import matplotlib.pyplot as plt
from scipy import fft, signal

1. Zeitsignal y bereitstellen und Abtastfrequenz fs bzw. Sampleanzahl N bestimmen#

data = np.loadtxt('data/messung.csv', delimiter=';', skiprows=1)
t = data[:,0]  # Zeitschritte
y = data[:,1]  # Messignal
N = len(t)  # Sampleanzahl
fs = 1 / (t[1]-t[0])  # Abtastfrequenz

2. Fensterfunktion anwenden#

# Fenstertyp 'boxcar' ist gleichbedeutend mit keiner Fensterung
w = signal.windows.get_window('tukey', N)  # Fensterfunktion
y_w = w * (y - y.mean()) + y.mean()  # Fensterfunktion anwenden (Offset für Fensterung entfernen)

3. FFT berechnen und Amplituden korrekt skalieren#

Y = fft.rfft(y_w)  # Fourier-Transformation berechnen
A = abs(Y)/N  # Amplituden
A[1:] = A[1:]*2  # Amplituden skalieren

4. Frequenzachse erzeugen#

f = fft.rfftfreq(N, 1/fs)  # Frequenzachse erstellen

5. Spektrum darstellen#

fig, axs = plt.subplots(2,1, figsize=(12,7))
axs[0].plot(t, y, 'c', label='Original')
axs[0].plot(t, y_w, 'b', label='Fensterung')
axs[0].set_xlabel('Zeit $t$ [s]')
axs[0].set_ylabel('$y$ [V]')
axs[0].grid()
axs[1].plot(f, A, 'r.-')
axs[1].set_xlabel('Frequenz $f$ [Hz]')
axs[1].set_ylabel('Amplitudenspektrum $\mid Y \mid$ [V]')
axs[1].grid()
plt.show()
