Unterrichtsbeispiel#


← Einführung

Lernziele#

  • Sie können eine Funktion mit mehreren Übergabeparametern und Defaultwerten erstellen.

  • Sie können Daten aus einem anderen Programm in Python importieren und damit ein Diagramm erstellen.

Aufgabe#

Wir wollen den folgenden Riementrieb regelungstechnisch analysieren und einen passenden Regler auswählen und parametrieren. Die Regelstrecke besteht aus Motor, Übersetzung und Riementrieb mit Schlitten.

  • Eingangssgrösse (Stellgrösse \(y\)): Spannung am Motor [V]

  • Ausgangsgrösse (Regelgrösse \(x\)): Schlittenposition [mm]

Für den Aufbau der Regelstrecke stehen folgende Informationen zur Verfügung:

  • Übersetzung: Antriebsseite ⌀ 20 mm / Abtriebsseite ⌀ 50 mm

  • Zahnriemen: Riemenscheiben ⌀ 30 mm

  • Motor: siehe untenstehender Datenblatt-Auszug

import numpy as np
import matplotlib.pyplot as plt

a) Regelstreckenparameter#

Bestimmen Sie den Typ und die Parameter der Regelstrecke anhand der obigen Informationen.

# Motor (P-T_1)
K_P_Motor = 173 / 60  # [1/(V*s)]
T_1_Motor = 0.0317  # [s]

# Übersetzung (P)
K_P_Uebersetzung = 20 / 50  # [-]

# Riemen & Schlitten (I)
K_I_Riemen = 30 * np.pi  # [mm]

# Gesamte Regelstrecke (I-T_1)
K_I = K_P_Motor * K_P_Uebersetzung * K_I_Riemen  # [mm/(V*s)]
T_1 = T_1_Motor

b) Reglerauslegung#

Wählen Sie einen passenden Regler und bestimmen Sie die Reglerparameter gemäss CHIEN, HRONES, RESWICK (vgl. untenstehende Tabelle). Implementieren Sie dazu die Funkton CHR(K_IS, T_e, type=..., overshoot=..., mode=...) wobei mit type der Reglertyp (P/PI/PID), mit overshoot (True/False) das Überschwingen und mit mode das gewünschte Verhalten (Führungs-/Störverhalten bzw. F/S) eingestellt werden kann.

def CHR(K_IS, T_e, type='PID', overshoot=True, mode='F'):
    K_PR, T_i, T_d = None, None, None
    if overshoot:  # mit Überschwingen
        if mode == 'S':  # Störverhalten
            if type == 'P':
                K_PR = 0.71 * 1/(K_IS * T_e)
            elif type == 'PI':
                K_PR = 0.71 * 1/(K_IS * T_e)
                T_i = 2.3 * T_e
            elif type == 'PID':
                K_PR = 1.2 * 1/(K_IS * T_e)
                T_i = 2.0 * T_e
                T_d = 0.42 * T_e
        elif mode == 'F':  # Führungsverhalten
            if type == 'P':
                K_PR = 0.71 * 1/(K_IS * T_e)
            elif type == 'PI':
                K_PR = 0.59 * 1/(K_IS * T_e)
            elif type == 'PID':
                K_PR = 0.95 * 1/(K_IS * T_e)
                T_d = 0.47 * T_e
    else:  # ohne Überschwingen
        if mode == 'S':  # Störverhalten
            if type == 'P':
                K_PR = 0.3 * 1/(K_IS * T_e)
            elif type == 'PI':
                K_PR = 0.59 * 1/(K_IS * T_e)
                T_i = 4.0 * T_e
            elif type == 'PID':
                K_PR = 0.95 * 1/(K_IS * T_e)
                T_i = 2.4 * T_e
                T_d = 0.42 * T_e
        elif mode == 'F':  # Führungsverhalten
            if type == 'P':
                K_PR = 0.3 * 1/(K_IS * T_e)
            elif type == 'PI':
                K_PR = 0.34 * 1/(K_IS * T_e)
            elif type == 'PID':
                K_PR = 0.59 * 1/(K_IS * T_e)
                T_d = 0.5 * T_e 
    return K_PR, T_i, T_d

CHR(K_I, T_1)  # Reglerparameter

(0.2757010191960405, None, 0.014898999999999999)

c) Führungssprungantwort#

Bauen Sie den kompletten Regelkreis in BORIS auf mit dem gewählten Regler und simulieren Sie die Führungssprungantwort (Simulationsdauer = 0.5 s, Zeitschritt = 0.001 s). Um ein ansprechendes Diagramm generieren zu können, exportieren wir die Zeitverläufe in ein Textfile und lesen dieses anschliessend hier im Jupyter-Notebook ein.

Für das Exportieren von Daten in BORIS stehen verschiedene Möglichkeiten zur Verfügung. Wir verwenden den Block Tabellen-Ausgabedatei (→ Senken). Standardmässig wird dort das Komma als Dezimalseparator für Fliesskommazahlen (in Deutschland üblich) verwendet. Stellen Sie dies auf den Punkt um.

data = np.loadtxt('data/BORIS.TXT', skiprows=1)
t, w, x = data.T

# hier mit Axes-Interface (alternativ mit Pyplot-Interface)
fig, ax = plt.subplots(figsize=(10, 5))
ax.plot(t, w, 'b', label='w')
ax.plot(t, x, 'r', label='x')
ax.set_title('Führungssprungantwort')
ax.set_xlabel('Zeit $t$ [s]')
ax.set_ylabel('Führungsgrösse $w$ bzw. Regelgrösse $x$ [-]')
ax.grid()
ax.legend()
plt.show()
